# Legacy Prompts Archive

This folder preserves prompt snippets from Toolbox version `0.0.1`.

They are no longer the active workflow for current implementation work, but they still provide useful historical context about:

- early product intent
- early UI copy
- initial tool ideas and constraints

Use this folder as reference material only.

For current work, prefer:

- `docs/` for the repository contract and standards
- `skills/` for the active repo-local Codex workflow

If a legacy prompt conflicts with a current document in `docs/`, treat `docs/` as the source of truth.
