# Component Checklist

- Confirm the component name is `PascalCase`.
- Confirm handlers and helpers use `camelCase`.
- Confirm the stylesheet name matches the component name.
- Confirm CSS classes use `kebab-case`.
- Confirm visible strings use i18n.
- Confirm the component does not hardcode theme colors in TSX.
- Confirm inline styles exist only for truly dynamic values.
- Confirm logic extraction was considered if the component became large or mixed-responsibility.
