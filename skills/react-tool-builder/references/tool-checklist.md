# Tool Checklist

- Choose a stable `kebab-case` tool id.
- Add metadata in the tool registry.
- Create one top-level view component.
- Extract logic to a helper file when the transformation is testable.
- Add SCSS near the tool and use semantic tokens.
- Add locale entries in every supported language.
- Add or update tests for the tool behavior.
- Confirm navigation and discovery come from the registry.
