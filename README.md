# 🧰 Toolbox

A collection of tools and templates to accelerate modern web development with **React**, **TypeScript**, and **Vite**.

## 🚀 What is Toolbox?

Toolbox is a **developer productivity kit** that provides:

- Ready-to-use **React + TypeScript + Vite templates**
- Pre-configured **ESLint**, **Jest**, and build setup
- **Modular architecture** for scalable and maintainable projects

Ideal for developers who want to start fast without sacrificing code quality.

## ⚙️ Tech Stack

- **React** – UI library for building interfaces
- **TypeScript** – Strongly typed superset of JavaScript
- **Vite** – Fast modern build tool
- **ESLint** – Keep your code clean and consistent
- **Jest** – Automated testing framework
